## Goal

### Notes on Estimated max walking time

* The size of the main level 1 is rougly: 2500px X 5500px box
* The entrance area is 600px deep area below the main level box
* Size of cafe is: 600px X 1100px
* Max manhattan distance: 2500 + 5500 + 600 + 600 + 1100 = 10300px
* Map scale: 1feet = 16px
* Walking speed: 4.5 feet / second
* Expected Max walk time = (10300 / 16 / 4.5) = 143s
